# Select

`Select` is a customizable and accessible input built as a drop-in replacement for `HTMLSelectElement`. The component allows users to select one or multiple options from a dropdown.

There is no `defaultValue` value prop here due to browser implementation. Use `<option selected>value</option>` instead.
