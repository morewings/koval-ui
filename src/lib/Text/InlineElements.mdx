# Inline Elements

The set of customizable and accessible inline text elements. These components include `A`, `B`, `Strong`, `I`, `Em`, `Mark`, `Small`, `Del`, `Ins`, `Sub`, `Sup`, `Kbd`, `Code`, and `S`. Each component supports additional properties for data attributes and library-specific props.
